# TayWeiChong.github.io

Source for my personal portfolio site — https://tayweichong.github.io

A static site: `index.html` plus an `images/` folder. No build step, no dependencies.

## Structure
- `index.html` — the page
- `images/` — dashboard replicas, maps, and samples used in the project details
- `.nojekyll` — serves files directly (no Jekyll processing)

## Hosting (GitHub Pages)
1. Repo must be named exactly `TayWeiChong.github.io`.
2. Keep `index.html` and the `images/` folder at the repository root.
3. Settings → Pages → Build and deployment → Source: Deploy from a branch → `main`.
4. Live at https://tayweichong.github.io (username lowercased).

## Editing
Edit `index.html` directly. To change an image, replace the file in `images/` keeping the same name. Changes go live a few minutes after you push.
